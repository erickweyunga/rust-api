# Rust API

[![Crates.io](https://img.shields.io/crates/v/rust-api.svg)](https://crates.io/crates/rust-api)
[![Documentation](https://docs.rs/rust-api/badge.svg)](https://docs.rs/rust-api)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)

A fast and flexible web framework for Rust that encourages rapid development and clean, pragmatic design.

## Documentation

All documentation is available at https://docs.rs/rust-api. If you're just getting started:

- First, read the installation instructions at https://docs.rs/rust-api
- Next, work through the examples in the `examples/` directory
- Read through the API reference for detailed information
- Check out the CHANGELOG.md for recent updates

## Getting Help

- Read the documentation at https://docs.rs/rust-api
- Open an issue at https://github.com/erickweyunga/rust-api/issues
